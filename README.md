# CODSOFT - Data Science Internship

This repository contains my project submissions for the CodSoft Data Science Internship.

## Tasks

### 1. Titanic Survival Prediction

Predicts passenger survival using the Titanic dataset.

- Cleaned missing values, encoded categorical features
- Trained a Random Forest Classifier
- **Accuracy:** 70%
- **Notebook:** `Titanic_dataset.ipynb`

### 2. Iris Flower Classification

Classifies iris flowers into species based on petal and sepal measurements.

- Used the classic Iris dataset from scikit-learn
- Trained a Random Forest Classifier
- **Accuracy:** ~97%
- **Notebook:** `Iris_Classification.ipynb`

### 3. Movie Rating Prediction

Predicts movie ratings based on genre, director, duration, and votes.

- Preprocessed categorical features with one-hot encoding
- Trained a Random Forest Regressor
- **R2 Score:** ~0.39
- **Notebook:** `Movie_Rating_Prediction.ipynb`

## Tools

Python, Pandas, Scikit-learn, Matplotlib, Google Colab
